## Getting Started ğŸš€

1. Clone the repository:
   ```bash
   git clone https://github.com/aryan1112003/AI-Powered-Decentralized-Marketplace
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Create a `.env` file:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   ```
